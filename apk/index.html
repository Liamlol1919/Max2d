Intro

Welcome to the APK building Tutorial. I wrote 2 Methods (Easy/ Hard) step by step down. The Easy Method can't have a custom Splash Screen, is more laggy and contains more bugs.

Method 1, Mobile

Required

Android 5+
File Exlorer
Install these Programs:
- Mechanix

Step 1

Open your File Explorer, Go to emulated/0/ Dictionary (Where Downloads, Documents and Android Folders are) and create a New Folder called "MechaX".

Step 2

Now go to your Projects Folder, which is located in the /Android, /data, /com.mikeytronix.., /files, /mobilegameengine, /projects Folder and Copy the Game Folder into the /MechaX

Step 3

Open the 2D MechaniX App, Install APKtool M, Enable File Acces , Re-open the App and Select your Project. Then Tap the "Build".

Step 4

Now you can customize your APK Icon, Name, Path and Version. Be sure you have a Path name (com.yourname.gamename) without special characters like -/[/?/;/> or spaces. When you are done, you can click "Build Now"

Step 5

Enable and skip all Pop-Ups from Apktool and click the .0Build[2Mechanix] folder. After that click the ▶️ Button and select aapt2 and Default Signature(Testkey). When you get an Error with yellow Text just close the Apktool App, go to Mechanix and Build again until it Works. Share the Apk by long pressing Flappy.apk.

Errors

- Game Freezes / Not Moving

  • Make sure you haven't deleted any Variable
  • Put Timer Delayed after every Event
  • Set Sprite Opacity to 1 or 0
- Game not Loading (Splash)
  • Use Method 2
  • Ask Liamdev
